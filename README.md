# Letta Telegram Bot

Multi-user Telegram bot for Letta cloud. This bot lets you provide your agents and agent architectures to end-users. Heavily inspired by <https://github.com/letta-ai/letta-telegram/>

## Flexibility & Contributions Welcome

This project is a research tool for rapidly testing agent architectures with real users. We're very flexible - feel free to open an issue or PR for:

- Different storage backends support (PostgreSQL, MongoDB, SQLite, etc.)
- Alternative deployment strategies (k8s, serverless, edge, etc.)
- Custom authorization flows (OAuth, payments, invites, etc.)
- Anything else that could be useful for your case
